⚔️ Projeto Jogo RPG
📝 Descrição
Este é um projeto de RPG (Role Playing Game) desenvolvido para exercitar conceitos fundamentais de Programação Orientada a Objetos (POO). O jogo funciona via consola, onde o jogador pode criar personagens, e evoluir as suas habilidades.

O foco principal do projeto é a aplicação de:

Abstração: Criação de classes base para heróis e monstros.

Herança: Especialização de classes (Guerreiro, Mago, Arqueiro).

Polimorfismo: Métodos de ataque e defesa que se comportam de forma diferente para cada classe.

Encapsulamento: Proteção dos atributos de vida, mana e experiência.

🎮 Funcionalidades
[x] Criação de Personagem: Escolha de nome e classe.

[x] Sistema de Combate: Ataques básicos, habilidades especiais
